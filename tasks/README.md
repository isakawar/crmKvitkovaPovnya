# Tasks

Тут зберігаються файли активних задач. Claude оновлює їх після реалізації.

## Формат файлу

Назва: `YYYY-MM-DD_short-description.md`

Приклад: `2026-04-15_add-courier-rating.md`

## Шаблон

```markdown
# Назва задачі

## Статус
in_progress | done

## Задача
Короткий опис що потрібно зробити.

## План
- [ ] Крок 1
- [ ] Крок 2

## Реалізація
Що було зроблено, які файли змінено.

## Як тестувати
1. Крок тестування
2. Очікуваний результат

## Edge cases
- ...
```
