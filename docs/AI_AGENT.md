# AI Агент — Документація

AI асистент вбудований у CRM і доступний через кнопку з іконкою робота у правому нижньому куті екрану (тільки для ролей `admin` і `manager`).

Агент розуміє природну українську мову, виконує дії через підтвердження та зберігає контекст розмови (3 дні).

---

## Можливості

### Читання даних (без підтвердження)

| Дія | Опис |
|-----|------|
| Пошук доставок | За датою, статусом, телефоном, Instagram |
| Пошук замовлень | За текстом, датою, телефоном, Instagram |
| Пошук клієнтів | За Instagram, телефоном, Telegram |
| Список кур'єрів | Активні кур'єри з телефонами |

### Зміна даних (з підтвердженням)

| Дія | Опис |
|-----|------|
| Перенести доставку | Змінити дату/час доставки |
| Оновити доставку | Змінити адресу, коментар, побажання |
| Змінити статус доставки | Очікує / Доставлено / Скасовано |
| Призначити кур'єра | На одну або кілька доставок |
| Створити клієнта | Instagram + телефон + (необов'язково) Telegram |
| Створити замовлення | Нове замовлення для існуючого клієнта |

> Перед будь-якою зміною агент показує картку підтвердження. Дані не змінюються до натискання **"Підтвердити"**.

## Довідка

### Доступні статуси доставок
- `Очікує` — заплановано
- `Розподілено` — призначено кур'єра
- `Доставлено` — виконано
- `Скасовано` — скасовано

### Розміри букетів
`S` / `M` / `L` / `XL` / `XXL` / `Власний`

### Формат дат і часу
- Дата: `17.04.2026` або `17 квітня` або `2026-04-17`
- Час: `14:00` або `з 12 до 14`

---

## Як це працює

1. Ви пишете запит у чат
2. Агент знаходить потрібні дані через пошук
3. Якщо потрібна зміна — агент готує **preview** і показує картку
4. Ви натискаєте **Підтвердити** або **Скасувати**
5. Підтверджена дія виконується і логується в базі

Картка підтвердження діє **10 хвилин** — після цього дія анулюється.

---

## Обмеження

- Агент не створює підписки (Weekly/Monthly/Bi-weekly) — тільки разові замовлення
- Агент не редагує замовлення — тільки доставки
- Чат доступний тільки для `admin` і `manager`
- Якщо знайдено кілька збігів — агент **не вибирає сам**, а просить уточнити
