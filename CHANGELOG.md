# Changelog

## [0.0.6] - 2025-23-07
### 🎉 **Великий релізи - CRM система та Telegram Bot інтеграція**

#### ✨ **Нові сторінки та функціонал:**
- **Повноцінна сторінка розподілу доставок** (`/distribution`) з drag & drop функціоналом
- **Сторінка управління кур'єрами** (`/couriers`) з можливістю додавання, редагування та управління статусами
- **Telegram Bot інтеграція** для автоматичних сповіщень кур'єрів про доставки
- **Реєстрація кур'єрів** через Telegram бота з підтвердженням від адміністратора

#### 🚚 **Розподіл доставок:**
- Інтерактивний drag & drop між кур'єрами та нерозподіленими доставками
- Швидке призначення з вибором кількох доставок та масовими операціями
- Візуалізація статусу надіслання Telegram повідомлень
- Копіювання адрес одним кліком
- Згортання/розгортання секцій кур'єрів
- Статистична панель з підрахунком доставок, кур'єрів та відправлених сповіщень
- Адаптивний дизайн для роботи на мобільних пристроях

#### 🤖 **Telegram Bot функціонал:**
- Автоматична реєстрація кур'єрів через команду `/start`
- Надсилання детальної інформації про доставки кур'єрам
- Підтвердження отримання доставок через інтерактивні кнопки
- Оновлення статусів доставок в реальному часі
- Зручний формат повідомлень з номерацією та всією необхідною інформацією

#### 👥 **Управління кур'єрами:**
- Список всіх кур'єрів з відображенням статусу Telegram реєстрації
- Додавання нових кур'єрів через веб-інтерфейс
- Активація/деактивація кур'єрів
- Інтеграція з Telegram для автоматичного зв'язування облікових записів

#### 🐛 **Виправлення помилок:**
- Виправлено проблеми з Docker контейнерами та перезапуском сервісів
- Оновлено deprecated SQLAlchemy API до сучасних версій
- Виправлено скидання статусу Telegram сповіщень при перепризначенні доставок
- Оптимізовано відображення та простір інтерфейсу

#### 🔧 **Технічні покращення:**
- Docker Compose конфігурація з окремими сервісами для веб-додатку та Telegram бота
- Міграції бази даних для підтримки Telegram функціоналу
- Environment configuration для зберігання токенів та налаштувань
- Логування та моніторинг для всіх сервісів

## [0.0.5] - 2025-07-22
- **Пофіксили баг з відображенням суми власного букету**: тепер в таблиці доставок для власного розміру показується сума (наприклад "2500₴") замість напису "Власний"
- **Додали фільтрацію за період**: можна фільтрувати доставки за діапазоном дат ("Дата з" / "Дата до") на додаток до існуючого фільтру по одній даті
- **Додали можливість видаляти доставки**: тепер можна видаляти доставки зі статусом "Не оплачена" через кнопку в модальному вікні редагування
- **Прибрали поле "дата першої доставки" для продовження підписки**: спрощено форму продовження підписки, видалено непотрібне поле
- Виправлено роботу фільтрів по фінансовому тижню після додавання фільтрації за період
- Додано поле для редагування власної суми в модальному вікні редагування доставки
